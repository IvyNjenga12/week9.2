Types of Object Properties

1. Data Properties

These properties hold values.

        let laptop = {
            brand: "Dell",
            price: 800,
            isNew: true
        };

        console.log(laptop.brand); // Dell
        console.log(laptop.price); // 800
        console.log(laptop.isNew); // true

2. Computed Properties (Dynamic Properties)

Properties can be set dynamically using square brackets.

        let key = "color";
        let bike = {
            brand: "Yamaha",
            [key]: "Black"
        };

        console.log(bike.color); // Black
